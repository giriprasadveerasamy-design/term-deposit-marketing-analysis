# Term Deposit Telemarketing Analysis

This repository contains a Power BI business case analyzing a bank’s term deposit telemarketing campaign using customer demographic and behavioral data.

## Files
TD_Term_Deposit_Telemarketing_Analysis_Report.pbix 
TD_Term_Deposit_Telemarketing_Analysis_Report.pdf  
 

## Objective
Identify customer segments most likely to subscribe to term deposits in order to improve the effectiveness and efficiency of telemarketing campaigns.

## Dataset
Bank Marketing Dataset (Kaggle) – `train.csv`
